# 🎂 Premium Birthday Wishes Website

A modern interactive birthday celebration website built using:

- HTML5
- CSS3
- JavaScript
- Three.js
- GSAP

## Features

- Animated Hero Section
- Typewriter Messages
- Gallery Lightbox
- Birthday Wishes Cards
- Countdown Timer
- Music Player
- Surprise Animation
- Gift Box Interaction
- Fireworks
- Floating Hearts
- Sparkles
- Mobile Responsive

index.html
