# Birthday Wishes Website Deployment Guide

## Local Development

1. Open project in VS Code
2. Install Live Server extension
3. Right click index.html
4. Click "Open with Live Server"

## Netlify Deployment

1. Login to Netlify
2. Click Add New Site
3. Deploy Manually
4. Drag and Drop project folder

## Vercel Deployment

1. Login to Vercel
2. Create New Project
3. Import Project Folder
4. Deploy

## GitHub Pages

1. Push project to GitHub
2. Open Repository Settings
3. Pages
4. Select Branch
5. Save

Website will be live in a few minutes.
